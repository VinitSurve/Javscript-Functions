Javascript Functions

Sr no. Function Description Example
1. parseInt() This function is to parse a string and convert it into a number Example                              Output
parseInt('MH100')               Nan
parseInt('100')                     100
parseInt('100MH')               100
2. parseFloat() This function is to parse a string and convert it into floating point representation Example                              Output
parseFloat('MH100')           Nan
parseFloat('100.00')            100
parseFloat('100.2MH')        100
3. alert() This fuction displays alert popup box with ok button.This is also called as a message box. Example
alert("Welcome to Javacscript") or window.alert("Hello")
4. prompt() This function is used when you want to input value from user at the time of program execution.It displays ok and cancel buttons.Ok button returns input value, Cancel button return null value. Example
var n;
n = prompt("Enter value")
5. confirm() This function displays confirmation message box with ok and cancel button.OK button value returns 'true' and cancel returns 'false' Example
var ans;
ans = confirm("Do you want to continue")
6. toLowerCase() This function is used to convert the given string into lowercase alphabets Example
var str= "Javscript"
str=str.toLowerCase()
7. toUpperCase() This function is used to convert the given string into Uppercase alphabets Example
var str="Javascript" str=str.toUpperCase()
8. isNan() It returns true "if given value is not a number".It returns false "if given value is a number". Example
isNan(CP100)
isNan(100)
